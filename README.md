# 📊 Bank Loan Analysis Dashboard (End-to-End Power BI Project)

## 📝 Project Overview
This project is an enterprise-grade financial analytics dashboard developed in Power BI to monitor and analyze bank loan performance, funding trends, and risk management. Designed for financial administrators and stakeholders, it provides actionable insights into loan distribution, borrower demographics, and the critical distinction between "Good" and "Bad" loan portfolios.

## 🛠️ Key Features Implemented

### 1️⃣ Data Modeling & Architecture 📐
*   **Star Schema Layout**: Established a robust data model with optimized relationships between loan application dimensions and financial fact tables.
*   **Streamlined Workspace**: Applied industry-grade naming conventions and organized fields for a clean, professional user experience.

### 2️⃣ Advanced Analytics & Performance Metrics 🧮
*   **KPI Development**: Utilized advanced DAX measures to compute Total Loan Applications, Total Funded Amounts, and Total Amounts Received. 
*   **Risk Assessment**: Engineered clear visualizations to differentiate between "Good Loan" (fully paid/current) and "Bad Loan" (charged off) issuances to help in rapid decision-making. 

### 3️⃣ Time-Intelligence & Sessional Trends ⏳
*   **Monthly Performance**: Tracked the "Total Amount Received by Month" to identify seasonal trends and growth patterns in loan repayments.
*   **Regional & Demographic Analysis**: Analyzed loan data across different states, employment lengths, and home ownership statuses (Mortgage, Rent, Own).

### 4️⃣ Dashboard Layout & Visualizations 🎨
The report architecture is divided into three comprehensive views:
*   **Summary Dashboard**: Provides an executive-level overview of key metrics, including average interest rates and Debt-to-Income (DTI) ratios. 
*   **Overview Dashboard**: Features deep-dive charts for trends by state, purpose, and loan terms.
*   **Details Dashboard**: A granular matrix view allowing for the tracking of individual loan records (ID, Purpose, Grade, Funded Amount, etc.).
